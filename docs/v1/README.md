# v1 Documentation

This folder contains documentation for JMeter Test Generator v1.x.

## Overview

v1 provides endpoint catalog generation from OpenAPI specifications:
- Scan projects for OpenAPI/Swagger specs
- Generate JMX files with HTTP samplers for all endpoints
- Validate existing JMX files
- Detect spec changes and update JMX files
